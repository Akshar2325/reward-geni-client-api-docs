Getting Started

Get up and running with the Reward Geni Client Exchange API in minutes. This guide walks you through authentication and making your first API call.

Prerequisites #

Before you begin, make sure you have the following:

  • Your cxClientId — provided when your client account is created
  • Your cxClientSecret — shown only once at creation time
  • Your cxApiSecret — secondary secret for server-to-server calls
  • Access to the staging or production environment
⚠️
Store credentials securely. The cxClientSecret and cxApiSecret are shown only once when your account is created. If lost, they must be rotated by a super admin.

Authentication Flow #

The API uses a two-step authentication process:

  1. Get Your Credentials

    When your client account is created, the Reward Geni admin receives:

    Field Description
    cxClientId Your Client Exchange ID (public identifier)
    cxClientSecret Your Client Exchange Secret (shown once only)
    cxApiSecret Secondary secret for server-to-server operations

  2. Generate an Access Token

    Exchange your credentials for a short-lived Bearer token (1 hour TTL).

    POST /clientapi/generate-token

    Send your cxClientId and cxClientSecret as headers:

    Headers 
    x-cx-client-id: <your-cxClientId>
x-cx-client-secret: <your-cxClientSecret>

    Example using different tools:

    cURL 
    curl -X POST https://staging.rewardgenix.co.in/clientapi/generate-token \
  -H "x-cx-client-id: YOUR_CLIENT_ID" \
  -H "x-cx-client-secret: YOUR_CLIENT_SECRET"
    JavaScript 
    const response = await fetch('https://staging.rewardgenix.co.in/clientapi/generate-token', {
  method: 'POST',
  headers: {
    'x-cx-client-id': 'YOUR_CLIENT_ID',
    'x-cx-client-secret': 'YOUR_CLIENT_SECRET'
  }
});

const data = await response.json();
console.log('Access Token:', data.data.accessToken);
    Node.js 
    const axios = require('axios');

const response = await axios.post(
  'https://staging.rewardgenix.co.in/clientapi/generate-token',
  {},
  {
    headers: {
      'x-cx-client-id': 'YOUR_CLIENT_ID',
      'x-cx-client-secret': 'YOUR_CLIENT_SECRET'
    }
  }
);

console.log('Access Token:', response.data.data.accessToken);

    Success response:

    JSON 
    {
  "message": "Access token generated successfully",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresIn": 3600,
    "expiresAt": "2025-01-01T13:00:00.000Z",
    "tokenType": "Bearer"
  }
}
    ℹ️
    Token rotation: Any previously active token for the same client is automatically revoked when a new one is generated.

  3. Use the Token in API Calls

    Include the Bearer token in the Authorization header for all subsequent requests:

    Header 
    Authorization: Bearer <accessToken>

Your First API Call #

Once you have an access token, try creating a user:

cURL 
curl -X POST https://staging.rewardgenix.co.in/clientapi/users \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "x-cx-api-secret: YOUR_API_SECRET" \
  -d '{
    "users": [{
      "userId": "EMP001",
      "fullName": "John Doe",
      "firstName": "John",
      "lastName": "Doe",
      "emailId": "john.doe@example.com",
      "phoneNumber": "9876543210"
    }]
  }'
JavaScript 
const response = await fetch('https://staging.rewardgenix.co.in/clientapi/users', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'x-cx-api-secret': 'YOUR_API_SECRET'
  },
  body: JSON.stringify({
    users: [{
      userId: 'EMP001',
      fullName: 'John Doe',
      firstName: 'John',
      lastName: 'Doe',
      emailId: 'john.doe@example.com',
      phoneNumber: '9876543210'
    }]
  })
});

const result = await response.json();
console.log(result);
Node.js 
const axios = require('axios');

const response = await axios.post(
  'https://staging.rewardgenix.co.in/clientapi/users',
  {
    users: [{
      userId: 'EMP001',
      fullName: 'John Doe',
      firstName: 'John',
      lastName: 'Doe',
      emailId: 'john.doe@example.com',
      phoneNumber: '9876543210'
    }]
  },
  {
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
      'x-cx-api-secret': 'YOUR_API_SECRET'
    }
  }
);

console.log(response.data);
