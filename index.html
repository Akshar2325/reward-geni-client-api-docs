Endpoints Overview #

Method Endpoint Description
POST /clientapi/generate-token Generate an access token
POST /clientapi/users Create users (batch)
PUT /clientapi/users Update users (batch)
GET /clientapi/users Get users with pagination

Your Credentials #

When your client account is created, the Reward Geni admin provides the following credentials:

Field Description
cxClientId Your Client Exchange ID (public identifier)
cxClientSecret Your Client Exchange Secret (shown once only)
cxApiSecret Secondary secret for server-to-server operations
⚠️
Important: Store the raw cxClientSecret and cxApiSecret securely. They are shown only once at creation time.

Error Codes #

Authentication Errors

Code Status Description
ER_401_101 401 Invalid or expired token
ER_401_102 401 Missing or invalid client headers
ER_401_103 401 Invalid x-cx-api-secret
ER_403_001 403 IP not whitelisted

Validation Errors

Code Status Description
ER_400_001 400 Missing required fields in request
ER_400_002 400 Invalid field format
ER_400_003 400 Empty users array
ER_400_004 400 Unknown status value

Rate Limit & Server Errors

Code Status Description
ERR_429_001 429 Rate limit exceeded
ER_500_001 500 Internal server error
